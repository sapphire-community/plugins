## Features

-   Fully ready for TypeScript!
-   Provides a Decorator-based API to make developing your routes easy!
-   Follows common REST standards.
-   Includes ESM ready entrypoint

## Installation

```sh
yarn add -D @sapphire/plugin-api
```
